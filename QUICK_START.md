# Быстрый запуск "Призрачная Память"

## Запуск за 30 секунд

1. **Откройте Unity** и загрузите проект
2. **Откройте сцену** `Assets/Scenes/SampleScene.unity`
3. **Создайте GameObject** с именем "GameStarter"
4. **Добавьте скрипт** `GameStarter` на этот объект
5. **Нажмите Play** - игра автоматически настроится!

## Управление
- **WASD / Стрелки**: Движение
- **Пробел**: Прыжок
- **R**: Перезапуск уровня

## Механика
- **Умираете** → появляется призрак, повторяющий ваши действия
- **Призраки** можно использовать как платформы и активаторы
- **Стратегия**: умирайте в нужных местах для создания полезных призраков

## Демо-уровень
- Зеленые платформы - безопасные
- Красные треугольники - смертельные шипы
- Красные круги - кнопки (активируются призраками)
- Коричневые квадраты - двери
- Серые круги - падающие камни
- Желтый квадрат - финиш

## Ожидаемый результат
Уровень пройден за 2-5 смертей с использованием призраков как инструментов!
