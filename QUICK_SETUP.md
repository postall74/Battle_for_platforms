# Быстрая настройка игры "Призрачная Память"

## Шаг 1: Настройка сцены (5 минут)

### 1.1 Создание игрока
1. Создайте пустой GameObject → назовите "Player"
2. Добавьте компоненты:
   - `SpriteRenderer` → создайте белый квадрат 1x1
   - `Rigidbody2D` → Constraints: Freeze Rotation Z
   - `BoxCollider2D` → Size: 1, 1
   - `PlayerController` скрипт
3. Создайте дочерний объект "GroundCheck" (пустой Transform)
4. В PlayerController назначьте:
   - Ground Layer: создайте слой "Ground"
   - Ground Check: перетащите GroundCheck
   - Move Speed: 5
   - Jump Force: 12

### 1.2 Создание призрака
1. Создайте пустой GameObject → назовите "Ghost"
2. Добавьте компоненты:
   - `SpriteRenderer` → тот же спрайт, что и у игрока
   - `Rigidbody2D` → Constraints: Freeze Rotation Z
   - `BoxCollider2D` → Is Trigger = true
   - `Ghost` скрипт
3. Создайте префаб из этого объекта

### 1.3 Настройка GameManager
1. Создайте пустой GameObject → назовите "GameManager"
2. Добавьте `GameManager` скрипт
3. Настройте:
   - Spawn Point: создайте пустой Transform в начале уровня
   - Ghost Prefab: перетащите префаб призрака
   - Max Ghosts: 3

### 1.4 Настройка камеры
1. Выберите Main Camera
2. Добавьте `CameraFollow` скрипт
3. Target: перетащите Player

## Шаг 2: Создание простого уровня (10 минут)

### 2.1 Платформы
- Создайте несколько объектов с `BoxCollider2D` и `SpriteRenderer`
- Назначьте им слой "Ground"
- Расположите их как платформы

### 2.2 Ловушки
- Создайте объект с `BoxCollider2D` (Is Trigger = true)
- Добавьте `DeathZone` скрипт
- Сделайте красным цветом

### 2.3 Кнопка и дверь
- Создайте кнопку: объект с `BoxCollider2D` (Is Trigger = true) + `Button` скрипт
- Создайте дверь: объект с `BoxCollider2D` + `Door` скрипт
- В кнопке назначьте дверь как target object

### 2.4 Финиш
- Создайте объект с `BoxCollider2D` (Is Trigger = true)
- Добавьте `FinishZone` скрипт
- Сделайте зеленым цветом

## Шаг 3: Настройка UI (5 минут)

### 3.1 Создание Canvas
1. Правый клик в Hierarchy → UI → Canvas
2. Добавьте к Canvas `GameUI` скрипт

### 3.2 UI элементы
1. Создайте Text для количества призраков
2. Создайте Text для инструкций
3. Создайте панель паузы с кнопками
4. Назначьте ссылки в GameUI скрипте

## Шаг 4: Тестирование (5 минут)

### 4.1 Проверка базовой механики
1. Запустите игру
2. Проверьте движение игрока (WASD/стрелки)
3. Проверьте прыжок (пробел)
4. Умрите в ловушке → должен появиться призрак

### 4.2 Проверка призраков
1. Призрак должен повторять ваши действия
2. Попробуйте встать на призрака
3. Проверьте активацию кнопки призраком

## Готово! 🎉

Теперь у вас есть работающая игра с уникальной механикой призраков!

## Дополнительные улучшения

### Добавление падающих камней
1. Создайте объект с `Rigidbody2D` и `BoxCollider2D`
2. Добавьте `FallingRock` скрипт
3. Создайте дочерний объект с триггером для активации

### Добавление врагов
1. Создайте объект с `Rigidbody2D` и `BoxCollider2D`
2. Добавьте `PatrolEnemy` скрипт
3. Настройте точки патруля

### Настройка InputManager
1. Создайте пустой GameObject
2. Добавьте `InputManager` скрипт
3. Настройте клавиши управления

## Советы по отладке

- Включите Gizmos в Scene View для визуализации зон
- Проверьте консоль Unity на ошибки
- Убедитесь, что все коллайдеры настроены правильно
- Проверьте, что призраки имеют Is Trigger = true
