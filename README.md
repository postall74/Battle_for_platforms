# Призрачная Память - 2D Платформер

Уникальная 2D игра-платформер, где смерть игрока создает призрака, который повторяет все действия предыдущей попытки. Призраки можно использовать как платформы, активаторы и подсказки для прохождения уровня.

## Основные механики

### 🎮 Управление
- **WASD / Стрелки** - Движение влево/вправо
- **Пробел** - Прыжок
- **ESC** - Пауза

### 👻 Система призраков
1. **Запись действий** - Игра записывает все действия игрока каждые 0.1 секунды
2. **Создание призрака** - При смерти игрока создается полупрозрачный призрак
3. **Воспроизведение** - Призрак точно повторяет все действия предыдущей попытки
4. **Взаимодействие** - Призраки можно использовать как:
   - **Платформы** - На них можно стоять и прыгать
   - **Активаторы** - Они могут нажимать кнопки и открывать двери
   - **Подсказки** - Показывают опасные места и правильные пути

## Настройка проекта

### 1. Создание игрока
1. Создайте пустой GameObject и назовите его "Player"
2. Добавьте компоненты:
   - `SpriteRenderer` (с простым спрайтом)
   - `Rigidbody2D` (Constraints: Freeze Rotation Z)
   - `BoxCollider2D`
   - `PlayerController` скрипт
3. Создайте дочерний объект "GroundCheck" (пустой Transform)
4. Настройте в PlayerController:
   - Ground Layer (создайте слой "Ground")
   - Ground Check (перетащите GroundCheck)
   - Настройте скорость движения и силу прыжка

### 2. Создание призрака
1. Создайте пустой GameObject и назовите его "Ghost"
2. Добавьте компоненты:
   - `SpriteRenderer` (тот же спрайт, что и у игрока)
   - `Rigidbody2D` (Constraints: Freeze Rotation Z)
   - `BoxCollider2D` (Is Trigger = true)
   - `Ghost` скрипт
3. Создайте префаб из этого объекта

### 3. Настройка GameManager
1. Создайте пустой GameObject и назовите его "GameManager"
2. Добавьте `GameManager` скрипт
3. Настройте:
   - Spawn Point (перетащите точку возрождения)
   - Ghost Prefab (перетащите префаб призрака)
   - Max Ghosts (рекомендуется 3-5)

### 4. Создание уровня

#### Платформы
- Создайте объекты с `BoxCollider2D` и `SpriteRenderer`
- Назначьте им слой "Ground"

#### Ловушки (DeathZone)
- Создайте объект с `BoxCollider2D` (Is Trigger = true)
- Добавьте `DeathZone` скрипт
- Настройте target tags: "Player"

#### Кнопки
- Создайте объект с `BoxCollider2D` (Is Trigger = true)
- Добавьте `Button` скрипт
- Настройте target object (дверь или другой объект)

#### Двери
- Создайте объект с `BoxCollider2D`
- Добавьте `Door` скрипт
- Настройте open offset (смещение при открытии)

#### Финиш
- Создайте объект с `BoxCollider2D` (Is Trigger = true)
- Добавьте `FinishZone` скрипт

#### Падающие камни
- Создайте объект с `Rigidbody2D` и `BoxCollider2D`
- Добавьте `FallingRock` скрипт
- Создайте дочерний объект с триггером для активации

### 5. Настройка UI
1. Создайте Canvas
2. Добавьте UI элементы:
   - Text для отображения количества призраков
   - Text для инструкций
   - Панель паузы с кнопками Restart и Quit
3. Добавьте `GameUI` скрипт к Canvas
4. Настройте ссылки на UI элементы

### 6. Настройка камеры
1. Выберите Main Camera
2. Добавьте простой скрипт следования за игроком или используйте Cinemachine

## Пример уровня

### Простой уровень для демонстрации механик:

1. **Стартовая платформа** - где появляется игрок
2. **Пропасть с шипами** - первая смерть, создает призрака-подсказку
3. **Кнопка на стене** - которую нужно активировать призраком
4. **Дверь** - открывается кнопкой
5. **Падающий камень** - требует синхронизации с призраком
6. **Финишная зона** - цель уровня

### Последовательность прохождения:
1. Игрок падает в пропасть → создается призрак
2. Призрак показывает путь к кнопке
3. Игрок использует призрака для активации кнопки
4. Дверь открывается
5. Игрок проходит через дверь
6. Призрак активирует падающий камень
7. Игрок синхронизируется с призраком для безопасного прохода
8. Достижение финиша

## Настройки производительности

### Оптимизация записи:
- `recordInterval` - интервал записи (0.1 сек = 10 FPS)
- `maxRecordedFrames` - максимальное количество кадров записи
- `maxGhosts` - максимальное количество призраков

### Рекомендуемые настройки:
- **Запись**: 10 FPS (0.1 сек)
- **Максимум кадров**: 300 (30 секунд записи)
- **Максимум призраков**: 3-5
- **Время жизни призрака**: 30 секунд

## Расширение функционала

### Возможные улучшения:
1. **Разные типы призраков** - в зависимости от причины смерти
2. **Эффекты призраков** - свечение, частицы, звуки
3. **Враги** - реагирующие на призраков
4. **Секреты** - требующие создания призраков в определенных местах
5. **Множественные уровни** - с разными механиками
6. **Система очков** - за минимальное количество смертей

### Добавление новых механик:
1. Создайте новый скрипт, наследуясь от MonoBehaviour
2. Реализуйте интерфейс ITriggerable если нужно
3. Добавьте взаимодействие с призраками через OnTriggerEnter2D
4. Настройте визуальные эффекты и звуки

## Устранение неполадок

### Частые проблемы:
1. **Призраки не создаются** - проверьте ссылку на префаб в GameManager
2. **Игрок не может стоять на призраках** - проверьте настройки коллайдеров
3. **Кнопки не работают** - убедитесь, что коллайдер настроен как триггер
4. **Запись не работает** - проверьте настройки recordInterval

### Отладка:
- Включите Gizmos в Scene View для визуализации зон
- Используйте Debug.Log для отслеживания событий
- Проверьте консоль Unity на наличие ошибок
