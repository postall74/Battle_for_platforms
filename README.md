# Игра "Призрачная Память" / "Эхо Прошлого"

2D платформер с уникальной механикой, где каждая смерть игрока оставляет полупрозрачного призрака, который повторяет все действия предыдущей попытки и может использоваться как физическая платформа или активатор триггеров.

## Основная механика

- **Смерть без фрустрации**: При смерти игрок мгновенно респавнится в начале уровня
- **Призраки прошлого**: Каждая попытка оставляет призрака, который точно повторяет все действия игрока
- **Физическое взаимодействие**: На призраков можно вставать как на платформы
- **Активация триггеров**: Призраки могут нажимать кнопки и активировать механизмы
- **Стратегия через смерть**: Игрок может намеренно умирать в нужных местах для создания полезных призраков

## Настройка проекта в Unity

### 1. Создание базовых объектов

#### Игрок (Player)
1. Создайте пустой GameObject и назовите его "Player"
2. Добавьте компоненты:
   - `Rigidbody2D` (Gravity Scale: 3, Freeze Rotation Z: true)
   - `BoxCollider2D` (размер примерно 1x1)
   - `SpriteRenderer` (любой спрайт или цветной квадрат)
   - Скрипт `PlayerController`
3. Установите тег "Player"
4. Поместите на слой "Default" или создайте отдельный слой "Player"

#### Камера
1. Выберите Main Camera
2. Добавьте скрипт `CameraFollow`
3. Перетащите игрока в поле Target
4. Настройте offset (рекомендуется: 0, 2, -10)

#### Система призраков
1. Создайте пустой GameObject "GhostSystem"
2. Добавьте скрипт `GhostSystem`
3. Создайте префаб призрака:
   - Скопируйте игрока
   - Назовите "GhostPrefab"
   - Уберите скрипт `PlayerController`
   - Добавьте скрипт `Ghost`
   - Установите тег "Ghost"
   - Сделайте SpriteRenderer полупрозрачным (Alpha: 0.5)
   - Установите Collider как Trigger
4. Перетащите префаб в поле Ghost Prefab в GhostSystem

### 2. Создание уровня

#### Платформы (Ground)
1. Создайте GameObject с SpriteRenderer и Collider2D
2. Установите слой "Ground" (создайте если нужно)
3. Коллайдер должен быть обычным (не триггер)

#### Ловушки смерти
1. Создайте GameObject "DeathTrap"
2. Добавьте компоненты:
   - `SpriteRenderer` (красный цвет)
   - `BoxCollider2D` (Is Trigger: true)
   - Скрипт `DeathTrap`
3. Установите тег "DeathTrap"

#### Падающие камни
1. Создайте GameObject "FallingRock"
2. Добавьте компоненты:
   - `SpriteRenderer` (серый цвет)
   - `BoxCollider2D`
   - Скрипт `FallingRock`
3. Настройте параметры в скрипте (скорость падения, дистанция активации)

#### Кнопки и двери
1. **Кнопка**:
   - GameObject с SpriteRenderer (красный/зеленый)
   - BoxCollider2D (Is Trigger: true)
   - Скрипт `ButtonTrigger`
   - Тег "Button"

2. **Дверь**:
   - GameObject с SpriteRenderer (серый)
   - BoxCollider2D (обычный)
   - Скрипт `Door`

3. Перетащите дверь в поле Target Door кнопки

#### Патрулирующий враг
1. Создайте GameObject "Enemy"
2. Добавьте компоненты:
   - `SpriteRenderer` (фиолетовый цвет)
   - `BoxCollider2D` (Is Trigger: true)
   - Скрипт `PatrolEnemy`
3. Создайте две пустые точки "PointA" и "PointB"
4. Перетащите их в соответствующие поля скрипта

#### Финиш
1. Создайте GameObject "Finish"
2. Добавьте компоненты:
   - `SpriteRenderer` (зеленый цвет)
   - `BoxCollider2D` (Is Trigger: true)
3. Установите тег "Finish"

### 3. Настройка слоев и тегов

#### Теги (Tags)
- Player
- Ghost
- DeathTrap
- Button
- Finish

#### Слои (Layers)
- Default (игрок, призраки, враги)
- Ground (платформы)
- UI

#### Настройка Physics2D
1. В PlayerController установите Ground Layer на слой "Ground"
2. Это позволит игроку определять землю для прыжков

### 4. UI Setup
1. Создайте Canvas
2. Добавьте Text элементы для инструкций и счетчика смертей
3. Создайте GameObject "GameUI" с скриптом `GameUI`
4. Привяжите Text элементы к соответствующим полям

## Пример простого уровня

```
Старт -> Шипы -> Кнопка -> Дверь -> Падающий камень -> Враг -> Финиш
```

### Логика прохождения:
1. **Первая смерть**: Игрок умирает от шипов, изучает расположение
2. **Вторая попытка**: Обходит шипы, но не может пройти дверь
3. **Третья смерть**: Умирает на кнопке, создавая призрака-активатора
4. **Четвертая попытка**: Призрак держит дверь открытой, игрок проходит
5. **Пятая смерть**: Умирает от падающего камня, изучает тайминг
6. **Шестая попытка**: Использует призрака как отвлечение для врага
7. **Успех**: Достигает финиша

## Параметры настройки

### PlayerController
- `moveSpeed`: Скорость движения (рекомендуется: 5)
- `jumpForce`: Сила прыжка (рекомендуется: 12)
- `recordingRate`: Частота записи действий (рекомендуется: 10)

### GhostSystem
- `maxGhosts`: Максимальное количество призраков (рекомендуется: 5)
- `ghostColor`: Цвет призраков (полупрозрачный белый)

### FallingRock
- `fallSpeed`: Скорость падения (рекомендуется: 8)
- `triggerDistance`: Дистанция активации (рекомендуется: 3)
- `resetDelay`: Время до сброса (рекомендуется: 2)

## Советы по дизайну уровней

1. **Первая смерть должна быть очевидной** - покажите игроку основную механику
2. **Каждая смерть должна давать новое знание** или инструмент для прохождения
3. **Начинайте просто** - первые препятствия должны требовать 1-2 смерти максимум
4. **Комбинируйте механики** - используйте призраков как платформы И активаторы
5. **Визуальная ясность** - игрок должен понимать что опасно, а что полезно

## Управление

- **WASD / Стрелки**: Движение
- **Пробел**: Прыжок
- **Цель**: Дойти до зеленого финиша

## Расширение функционала

Для дальнейшего развития можно добавить:
- Разные типы призраков в зависимости от причины смерти
- Резонансные эффекты при синхронных действиях призраков
- Враги, по-разному реагирующие на призраков
- Систему улучшений между уровнями
- Более сложные головоломки с несколькими призраками
