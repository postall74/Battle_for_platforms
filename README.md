# Призрачная Память / Эхо Прошлого

2D платформер с уникальной механикой: когда игрок умирает, его предыдущая попытка остается в мире как полупрозрачный призрак, который повторяет все действия до момента смерти. Призраки можно использовать как платформы, активаторы кнопок и отвлекатели.

## Механики игры

### Основная концепция
- **Смерть создает призрака**: При смерти игрок возрождается, а его предыдущая попытка остается как призрак
- **Призраки повторяют действия**: Призраки точно воспроизводят все движения игрока до момента смерти
- **Физическое взаимодействие**: На призраков можно вставать, они активируют кнопки, отвлекают врагов
- **Стратегия через смерть**: Игрок может намеренно умирать в определенных местах для создания полезных призраков

### Управление
- **WASD / Стрелки**: Движение влево/вправо
- **Пробел**: Прыжок
- **R**: Перезапуск уровня

## Быстрый запуск (Автоматическая настройка)

### 1. Автоматическая настройка
1. Откройте сцену SampleScene
2. Создайте пустой GameObject и назовите его "GameStarter"
3. Добавьте скрипт `GameStarter`
4. Нажмите Play - игра автоматически настроится и запустится!

### 2. Ручная настройка (если нужна)

#### Создание игрока
1. Создайте пустой GameObject и назовите его "Player"
2. Добавьте компоненты:
   - `SpriteRenderer` (белый квадрат)
   - `Rigidbody2D` (Gravity Scale: 1)
   - `BoxCollider2D`
   - `PlayerController` скрипт
3. Создайте дочерний объект "GroundCheck" (пустой) и назначьте его в PlayerController
4. Установите тег "Player"

#### Создание призрака
1. Создайте пустой GameObject и назовите его "Ghost"
2. Добавьте компоненты:
   - `SpriteRenderer` (синий полупрозрачный квадрат)
   - `BoxCollider2D`
   - `GhostController` скрипт
3. Установите тег "Ghost"
4. Сделайте префаб и назначьте в PlayerController

#### Создание уровня

##### Платформы (Ground)
1. Создайте GameObject с `SpriteRenderer` и `BoxCollider2D`
2. Установите слой "Ground"
3. Установите тег "Ground"

##### Кнопки
1. Создайте GameObject с `SpriteRenderer` и `BoxCollider2D` (Is Trigger: true)
2. Добавьте `ButtonController` скрипт
3. Установите тег "Button"
4. Назначьте целевой объект (дверь или платформу)

##### Двери
1. Создайте GameObject с `SpriteRenderer` и `BoxCollider2D`
2. Добавьте `DoorController` скрипт
3. Установите тег "Door"

##### Шипы (ловушки)
1. Создайте GameObject с `SpriteRenderer` и `BoxCollider2D` (Is Trigger: true)
2. Добавьте `SpikeTrap` скрипт
3. Установите тег "DeathZone"

##### Падающие камни
1. Создайте GameObject с `SpriteRenderer`, `Rigidbody2D` и `BoxCollider2D`
2. Добавьте `FallingRock` скрипт
3. Установите тег "DeathZone"

##### Финиш
1. Создайте GameObject с `BoxCollider2D` (Is Trigger: true)
2. Установите тег "Finish"

#### Настройка камеры
1. Выберите Main Camera
2. Добавьте `CameraFollow` скрипт
3. Назначьте Player как target

#### Настройка UI
1. Создайте Canvas
2. Добавьте Text для счетчика смертей
3. Добавьте Text для инструкций
4. Создайте панель завершения уровня с кнопками

#### Настройка LevelManager
1. Создайте пустой GameObject "LevelManager"
2. Добавьте `LevelManager` скрипт
3. Назначьте все UI элементы и точки спавна/финиша

## Пример уровня

Создайте простой уровень с:
1. **Стартовая платформа** - где появляется игрок
2. **Шипы** - очевидная ловушка для демонстрации механики
3. **Кнопка + дверь** - показывают, как призраки активируют механизмы
4. **Падающий камень** - требует синхронизации с призраком
5. **Финиш** - цель уровня

## Советы по балансировке

1. **Первые ловушки должны быть очевидны** - игрок должен понять механику
2. **Призраки должны быть полезны** - каждый призрак должен давать новое знание/инструмент
3. **Уровень должен быть пройден за 2-5 смертей** для прототипа
4. **Визуальная обратная связь** - четкие цвета для разных типов объектов

## Возможные улучшения

- Разные типы призраков (в зависимости от причины смерти)
- Враги, реагирующие на призраков
- Перманентные улучшения
- Случайность в ловушках
- Эффекты "резонанса" при синхронных действиях призраков

## Тестирование игры

### Демо-уровень
Автоматически созданный уровень включает:
1. **Стартовая платформа** - игрок появляется здесь
2. **Шипы** - очевидные ловушки для демонстрации механики призраков
3. **Кнопка + дверь** - показывают, как призраки активируют механизмы
4. **Падающий камень** - требует синхронизации с призраком
5. **Финиш** - желтый квадрат, цель уровня

### Как играть
1. **Первая попытка**: Попробуйте пройти уровень, умерев на шипах
2. **Изучение призрака**: Ваш призрак покажет, где вы умерли
3. **Использование призрака**: Используйте призрака как платформу или активатор
4. **Стратегия**: Намеренно умирайте в нужных местах для создания полезных призраков

### Ожидаемый результат
- Уровень должен быть пройден за 2-5 смертей
- Каждая смерть должна давать новое знание/инструмент
- Призраки должны быть полезны для прохождения

## Технические детали

- **Запись действий**: каждые 0.1 секунды
- **Максимум призраков**: 5 одновременно
- **Физика призраков**: кинематические тела с коллайдерами
- **Платформенная механика**: проверка позиции игрока относительно призрака
