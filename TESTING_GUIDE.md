# Руководство по тестированию игры "Призрачная Память"

## Быстрый тест основной механики

### Минимальная тестовая сцена
1. **Игрок**: Квадрат с PlayerController
2. **Платформа**: Прямоугольник на слое Ground
3. **Ловушка**: Красный квадрат с DeathTrap
4. **Финиш**: Зеленый квадрат с FinishZone
5. **GhostSystem**: Пустой объект с системой призраков

### Проверка базового функционала

#### 1. Движение и прыжки
- [ ] Игрок движется влево/вправо (WASD/стрелки)
- [ ] Игрок прыгает (пробел) только на земле
- [ ] Камера следует за игроком
- [ ] Физика работает корректно (гравитация, коллизии)

#### 2. Система смерти
- [ ] Игрок умирает при касании ловушки
- [ ] Игрок мгновенно респавнится в начальной точке
- [ ] Счетчик смертей увеличивается

#### 3. Система призраков
- [ ] При смерти появляется полупрозрачный призрак
- [ ] Призрак точно повторяет движения предыдущей попытки
- [ ] На призрака можно встать как на платформу
- [ ] Призрак останавливается в месте смерти

#### 4. Завершение уровня
- [ ] При касании финиша уровень завершается
- [ ] Появляется сообщение о победе

## Тестирование сложных механик

### Система кнопок и дверей
1. Создайте кнопку и дверь
2. Свяжите их через Target Door
3. Проверьте:
   - [ ] Игрок активирует кнопку
   - [ ] Призрак активирует кнопку
   - [ ] Дверь открывается/закрывается
   - [ ] Коллизии двери работают правильно

### Падающие камни
1. Разместите FallingRock над игроком
2. Проверьте:
   - [ ] Камень падает при приближении игрока
   - [ ] Камень падает при приближении призрака
   - [ ] Камень убивает игрока при падении
   - [ ] Камень сбрасывается через заданное время

### Патрулирующие враги
1. Создайте врага с двумя точками патрулирования
2. Проверьте:
   - [ ] Враг движется между точками
   - [ ] Враг отвлекается на призраков
   - [ ] Враг убивает игрока при касании
   - [ ] Враг возвращается к патрулированию

## Отладка распространенных проблем

### Призраки не появляются
**Возможные причины:**
- Не настроен GhostSystem
- Отсутствует префаб призрака
- Не подписан на событие OnPlayerDied

**Решение:**
```csharp
// В GhostSystem.Start()
player = FindObjectOfType<PlayerController>();
if (player != null)
{
    player.OnPlayerDied += CreateGhost;
}
```

### Игрок проваливается сквозь призраков
**Причина:** Коллайдер призрака настроен неправильно

**Решение:**
- Призрак должен иметь Collider2D с isTrigger = true
- В PlayerController должна быть проверка призраков в CheckGrounded()

### Призраки не активируют кнопки
**Причина:** Призрак не обрабатывает триггеры

**Решение:**
Убедитесь что в Ghost.cs есть методы:
```csharp
void OnTriggerEnter2D(Collider2D other)
void OnTriggerStay2D(Collider2D other)  
void OnTriggerExit2D(Collider2D other)
```

### Камера дергается
**Причина:** Неправильная настройка CameraFollow

**Решение:**
- Используйте LateUpdate() для движения камеры
- Настройте followSpeed (рекомендуется 3-5)
- Проверьте offset камеры

### Игрок не может прыгать
**Возможные причины:**
- Неправильная проверка земли
- Слой Ground не настроен
- Rigidbody2D настроен неправильно

**Решение:**
- Проверьте groundLayer в PlayerController
- Убедитесь что платформы на слое Ground
- Rigidbody2D: Gravity Scale = 3, Freeze Rotation Z = true

## Тестирование геймплея

### Сценарий 1: Базовое обучение
1. Игрок видит очевидную ловушку
2. Умирает от неё (первый призрак)
3. Обходит ловушку, используя знание от призрака
4. Достигает финиша

### Сценарий 2: Призрак как платформа
1. Разместите недостижимую платформу
2. Игрок должен умереть в правильном месте
3. Использовать призрака как ступеньку
4. Достичь цели

### Сценарий 3: Призрак как активатор
1. Кнопка в опасном месте
2. Игрок умирает на кнопке
3. Призрак держит дверь открытой
4. Игрок проходит через дверь

### Сценарий 4: Комбинированная головоломка
1. Несколько препятствий подряд
2. Требуется 3-4 смерти для решения
3. Каждый призрак выполняет свою роль
4. Финальная попытка использует всех призраков

## Параметры для балансировки

### Скорости и силы
- **moveSpeed**: 3-7 (комфортное управление)
- **jumpForce**: 10-15 (достаточно для преодоления препятствий)
- **recordingRate**: 8-12 (плавность призраков)

### Тайминги
- **resetDelay** (FallingRock): 1-3 секунды
- **distractionDuration** (PatrolEnemy): 2-4 секунды
- **followSpeed** (CameraFollow): 2-5

### Дистанции
- **triggerDistance** (FallingRock): 2-4 единицы
- **detectionRange** (PatrolEnemy): 2-5 единиц

## Чек-лист финального тестирования

- [ ] Все базовые механики работают
- [ ] Призраки создаются и воспроизводятся правильно
- [ ] Физическое взаимодействие с призраками работает
- [ ] Триггеры активируются призраками
- [ ] UI отображается корректно
- [ ] Управление отзывчивое
- [ ] Уровень можно пройти за 5-10 смертей
- [ ] Каждая смерть дает новое знание
- [ ] Визуально понятно что опасно, а что полезно
- [ ] Нет критических багов или зависаний

## Советы по оптимизации

1. **Ограничьте количество призраков** (maxGhosts = 5)
2. **Очищайте старые записи** при достижении лимита
3. **Используйте Object Pooling** для призраков при необходимости
4. **Оптимизируйте частоту записи** (recordingRate = 10)
5. **Отключайте неиспользуемые компоненты** у призраков

Удачи в тестировании!
